## Notes
Description based on volumes: 1896-1897, 1900, 1910, 1920, 1930. For the year 1897, volume 1896-1897 contains the following information : “Conto corrente colla Posta. Camera di Comercio ed arti di Genova: Corso legale accertato dal sindacato degli Agenti di Cambio”. In 1930, the following sub-title appears: “Compilato dal Sindicato Agenti di Cambio“.
